# guestbook-app

## Projektbeschreibung
Eine einfache Gästebuch-Webanwendung. Benutzer können ihren Namen eintragen, der in einer Redis-Datenbank gespeichert und auf der Startseite angezeigt wird.

## Voraussetzungen
- Docker
- Docker Compose

## Startanleitung
1. Repository klonen:
```bash
   git clone https://github.com/vazgen96/guestbook-app.git
   cd guestbook-app
```

2. Anwendung starten:
```bash
   docker compose up
```

3. Im Browser öffnen:
```
   http://localhost:5000
```

## Verwendete Technologien
- Python 3
- Flask
- Redis
- Docker
- Docker Compose
